    JET meter bar

    A meter bar is typically used to display or accept user feedback on a product or service. 
    <oj-c-meter-bar value='4'></oj-c-meter-bar>

    Accessibility

    The application is strongly encouraged, even though it is not required, to populate the aria-label attribute on the element with meaningful descriptors as the oj-c-meter-bar element does not provide a default descriptor.

    If an application is using thresholds or reference lines, they need to include the information for screender readers using the following methods:

    • aria-describedby
    • aria-labelledby
    • aria-label

    If you are using indicatorColor, applications are responsible for making sure that the color meets the minimum contrast ratio.

    Keyboard

    Key Action
    Enter Submit the current value of the meter bar.
    Tab Move focus to next element and submit the current value of the meter bar.
    Shift + Tab Move focus to previous element.
    End Increase the meter bar's transient value to max value. Value is set after using Enter or Tab to submit.
    Home Decrease the meter bar's transient value to zero. Value is set after using Enter or Tab to submit.
    UpArrow Increase the meter bar's transient value. Value is set after using Enter or Tab to submit.
    DownArrow Decrease the meter bar's transient value. Value is set after using Enter or Tab to submit.
    LeftArrow Decrease the meter bar's transient value in left-to-right locales. Increase the meter bar's transient value in right-to-left locales. Value is set after using Enter or Tab to submit.
    RightArrow Increase the meter bar's transient value in left-to-right locales. Decrease the meter bar's transient value in right-to-left locales. Value is set after using Enter or Tab to submit.

    Touch End User Information

    Gesture Action
    Drag Value change when readonly is false.
